# Sprint 5 - Web Dashboard Checklist

## Goal
Build the React clinic-facing MVP.

## Screens
- [ ] Login screen.
- [ ] Clinic overview.
- [ ] Multi-dog monitoring board.
- [ ] Dog detail page.
- [ ] Latest telemetry table.
- [ ] Stress classification timeline or list.
- [ ] Alert queue.
- [ ] Vet notes/review section.
- [ ] Supplementary media review, if included.

## UX Checks
- [ ] Vet can see multiple dogs at a glance.
- [ ] High-stress cases are visually prioritized.
- [ ] Latest readings and timestamps are obvious.
- [ ] Device offline status is visible.
- [ ] Alert acknowledgement is simple.

## Exit Criteria
- [ ] Dashboard displays multiple test dogs.
- [ ] Dashboard displays latest telemetry.
- [ ] Dashboard displays stress status.
- [ ] Dashboard supports alert acknowledgement.
