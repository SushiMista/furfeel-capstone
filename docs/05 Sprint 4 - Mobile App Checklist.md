# Sprint 4 - Mobile App Checklist

## Goal
Build the Flutter owner-facing MVP.

## Screens
- [ ] Login screen.
- [ ] Dog list or selected dog screen.
- [ ] Dog status overview.
- [ ] Latest telemetry summary.
- [ ] Stress status display.
- [ ] Alert list.
- [ ] Alert detail.
- [ ] Health history preview.
- [ ] Supplementary media submission, if included.

## UX Checks
- [ ] Owner can understand the dog’s current status quickly.
- [ ] Stress labels are clear and not overly clinical.
- [ ] Alerts include recommended next action or review prompt.
- [ ] Offline/loading/error states are visible.
- [ ] Videos are labeled as supplementary assessment material.

## Exit Criteria
- [ ] Mobile app can show one dog’s latest telemetry.
- [ ] Mobile app can show one dog’s stress level.
- [ ] Mobile app can show alerts.
- [ ] Mobile app connects to Supabase securely.

## Evidence
- [ ] Mobile status screenshot.
- [ ] Alert screenshot.
- [ ] Health history screenshot.
