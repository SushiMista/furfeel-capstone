# Testing Strategy

## Test Layers

### Hardware Testing
- Sensor reading stability.
- Device connectivity.
- Power behavior.
- Harness comfort and placement.

### Backend Testing
- Telemetry intake.
- Validation rules.
- Database writes.
- API permissions.
- Alert creation.

### AI Testing
- Classification accuracy.
- Confusion matrix.
- False alerts.
- Inference latency.

### App Testing
- Mobile status display.
- Dashboard multi-dog monitoring.
- Alert acknowledgement.
- History and reports.

### ISO/IEC 25010 Mapping
- Functional suitability.
- Performance efficiency.
- Compatibility.
- Usability.
- Reliability.
- Security.
